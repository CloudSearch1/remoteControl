## 🚀 使用方式

### 启动服务器（被控电脑）

```bash
cd F:\remote-desktop
start-server.bat
```

**记录显示的 IP 地址**，例如：`192.168.1.100:5000`

### 启动客户端（控制电脑）

```bash
cd F:\remote-desktop
start-client.bat
```

**输入**:
- 服务器地址：`http://192.168.1.100:5000`
- 密码：`remote123`

## 🔐 安全提示

### 必须做的
1. ✅ **修改默认密码**
   ```bash
   SERVER_PASSWORD=your_strong_password
   ```

2. ✅ **防火墙配置**
   - 只开放必要端口
   - 限制访问 IP

3. ✅ **使用复杂密码**
   - 至少 12 位
   - 包含字母、数字、符号

### 可选的
- [ ] 配置 IP 白名单
- [ ] 启用 SSL/HTTPS
- [ ] 添加双因素认证
- [ ] 记录访问日志

## 📈 性能参考

### 局域网（推荐）
- **延迟**: < 100ms
- **帧率**: 10-20 FPS
- **带宽**: 1-5 Mbps

### 互联网
- **延迟**: 100-500ms
- **帧率**: 5-10 FPS
- **带宽**: 500K-2 Mbps

### 优化建议
1. 降低图像质量
2. 增大截图间隔
3. 使用有线网络
4. 关闭其他占用带宽的程序
