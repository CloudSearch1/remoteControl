# 远程桌面 - 快速启动指南

> **无需复杂依赖！开箱即用！**

---

## 🚀 立即开始

### 服务器端（被控电脑）

**步骤 1: 启动服务器**

双击运行：
```
F:\remote-desktop\start-simple.bat
```

**步骤 2: 记录地址**

服务器会显示：
```
本地 IP: 192.168.1.100
端口：5000
密码：remote123
```

**步骤 3: 访问控制页面**

在浏览器打开：
- 本地：`http://localhost:5000`
- 远程：`http://192.168.1.100:5000`

---

### 客户端（控制电脑）

**无需安装任何东西！**

1. 打开浏览器（Chrome/Edge/Firefox）
2. 输入服务器地址：`http://服务器 IP:5000`
3. 输入密码：`remote123`
4. 点击"连接"
5. 开始远程控制！

---

## 🎮 使用说明

### 鼠标控制
- **移动**: 在浏览器窗口移动鼠标
- **左键**: 点击
- **右键**: 右键点击
- **滚轮**: 滚动（部分浏览器支持）

### 键盘控制
- **打字**: 直接在浏览器窗口输入
- **特殊键**: 支持 Enter、Esc 等

### 系统监控
- **CPU**: 实时显示使用率
- **内存**: 实时显示使用率
- **磁盘**: 实时显示使用率

## ⚡ 特点

### 优点
- ✅ **零依赖** - 只需要 Python 3
- ✅ **网页控制** - 无需安装客户端
- ✅ **跨平台** - 任何浏览器都能用
- ✅ **简单快速** - 一键启动

### 限制
- ⚠️ 如果没有安装 pyautogui，鼠标键盘控制不可用
- ⚠️ 如果没有安装 Pillow，屏幕捕获使用替代方案

## 🔧 可选优化


## 🔐 修改密码

编辑 `server_simple.py`，找到这行：
```python
PASSWORD = 'remote123'
```

改成你的密码：
```python
PASSWORD = 'your_password'
```

重启服务器生效。

## 🌐 远程访问

### 局域网

1. 确保防火墙开放 5000 端口
2. 获取服务器 IP：`ipconfig`
3. 客户端访问：`http://服务器 IP:5000`

### 外网（需要端口转发）

1. 路由器设置端口转发：5000 → 服务器内网 IP:5000
2. 获取公网 IP
3. 客户端访问：`http://公网 IP:5000`

## ❓ 常见问题

### Q: 黑屏/无法显示？
**A**: 
- 检查服务器是否启动
- 检查浏览器是否支持 JavaScript
- 尝试刷新页面

### Q: 控制没反应？
**A**:
- 检查是否点击了"连接"
- 检查密码是否正确
- 如果没安装 pyautogui，控制功能不可用

### Q: 很卡顿？
**A**:
- 降低浏览器窗口大小
- 关闭其他占用带宽的程序
- 确保网络连接稳定
