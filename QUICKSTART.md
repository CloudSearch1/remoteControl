## 🚀 快速开始

### 服务器端（被控电脑）

**步骤 1: 安装依赖**
```bash
cd F:\remote-desktop
pip install -r requirements-server.txt
```

**步骤 2: 配置服务器**
```bash
# 编辑 .env 文件
# 修改密码和端口
```

**步骤 3: 启动服务器**
```bash
# 双击运行
start-server.bat

# 或命令行
python server.py
```

**步骤 4: 记录 IP 地址**
服务器启动后会显示本地 IP，例如：
```
本地 IP: 192.168.1.100
监听地址：0.0.0.0:5000
```

---

### 客户端（控制电脑）

**步骤 1: 安装依赖**
```bash
cd F:\remote-desktop
pip install -r requirements-client.txt
```

**步骤 2: 启动客户端**
```bash
# 双击运行
start-client.bat

# 或命令行
python client.py
```

**步骤 3: 连接服务器**
1. 输入服务器地址：`http://192.168.1.100:5000`
2. 输入密码：`remote123`（或你在 .env 中设置的密码）
3. 点击"连接"

## 🔧 配置说明

### 服务器配置 (.env)

```bash
# 监听地址
SERVER_HOST=0.0.0.0  # 所有网卡
SERVER_PORT=5000     # 端口

# 密码
SERVER_PASSWORD=remote123

# 图像质量 (1-100)
IMAGE_QUALITY=80

# 截图间隔 (秒)
SCREEN_CAPTURE_INTERVAL=0.1  # 10 FPS
```

### 客户端配置

客户端无需配置，连接时输入服务器地址和密码即可。

## 🌐 网络配置

### 局域网连接

**服务器端**:
1. 确保防火墙开放 5000 端口
2. 记录本地 IP（如 192.168.1.100）

**客户端**:
1. 输入服务器 IP 和端口
2. 连接即可

### 外网连接（需要端口转发）

**路由器配置**:
1. 登录路由器管理界面
2. 端口转发：5000 → 服务器内网 IP:5000
3. 获取公网 IP

**客户端**:
1. 输入公网 IP 和端口
2. 连接即可

## ❓ 常见问题

### Q: 连接失败？
**A**: 检查：
1. 服务器是否启动
2. 防火墙是否开放端口
3. IP 地址是否正确

### Q: 画面卡顿？
**A**: 尝试：
1. 降低图像质量（.env 中 IMAGE_QUALITY）
2. 增大截图间隔（SCREEN_CAPTURE_INTERVAL）
3. 检查网络带宽

### Q: 控制延迟？
**A**: 确保：
1. 网络连接稳定
2. 服务器和客户端在同一网络
3. 减少网络干扰

## 🔐 安全建议

1. **修改默认密码**
   ```bash
   SERVER_PASSWORD=your_strong_password
   ```

2. **使用复杂密码**
   - 至少 12 位
   - 包含大小写字母、数字、符号

3. **限制 IP 访问**（高级）
   在服务器代码中添加 IP 白名单

4. **使用 HTTPS**（高级）
   配置 SSL 证书加密传输

## 📊 性能优化

### 图像质量 vs 性能

| 质量 | 带宽 | 延迟 | 适用场景 |
|------|------|------|----------|
| 90-100 | 高 | 高 | 局域网、高速网络 |
| 70-89 | 中 | 中 | 一般网络 |
| 50-69 | 低 | 低 | 远程、慢速网络 |

```bash

```

```bash

```
